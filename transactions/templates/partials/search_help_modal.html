Quick Start Tips
Date Range

Filter transactions by specific time periods. Leave blank to search all dates.

Amount Range

Find transactions within specific dollar amounts. Uses absolute values.

Category

Filter by spending categories like Dining, Groceries, or Transportation.

Description

Search transaction descriptions for keywords like store names or notes.
