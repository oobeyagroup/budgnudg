How Keyword Rules Work

Priority Order:

  1. Keyword rules (highest priority)
  2. Learned patterns from "Learn from Current"
  3. AI merchant identification
  4. Default categorization

Examples:

  • "county" → Housing/Property Tax
  • "mortgage" → Housing/Mortgage
  • "gas" → Transportation/Gas
Pro Tip: Use higher priority values (e.g., 900) for very specific keywords, and lower values (e.g., 100) for more general ones.
